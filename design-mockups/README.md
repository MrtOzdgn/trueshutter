# TrueShutter design mockups

Non-functional, self-contained HTML concepts for the TrueShutter homepage. Each file is a single `.html` document — inline CSS, no external fonts/CDN/scripts, no build step. Open any of them directly in a browser. The idle/result toggle in each tool panel is pure CSS (radio-button hack), no JavaScript required.

## Round 1

### 01 — Instrument Panel
A cool near-black technical console: monospace type throughout, an amber accent, an aperture-blade dial mark in the header, and a glowing LED-style "odometer" readout for the shutter count. Screws, panel seams, and a faint diagonal hatch texture push it toward hardware/instrumentation rather than software. Personality: precise, mechanical, slightly retro-avionics.

### 02 — Darkroom
A warm amber-and-charcoal theme built on an upright/italic serif wordmark, film-grain texture, and a hand-set "exposure" layout for the result state (large serif numeral, brass corner marks). Personality: crafted, editorial, a little nostalgic — the warmest and most tactile of the six.

*Feedback: neither concept landed. Both are dark-first and fairly restrained; round 2 was asked to take more risk and cover more range.*

## Round 2

### 03 — Spec Sheet
**Light**, paper-white background with a faint graph-paper grid, tight black grotesk headlines, monospace data labels, and a single signal-red accent used sparingly (a period, an underline rule, a status square). The result state reads like a measurement certificate: a big black tabular number with a red underline, then a key/value spec list with dotted leader lines borrowed from a table of contents. Personality: precise and confident through rigor, not mood — the opposite pole from both round-1 concepts' darkness. Distinct from 05 (also light) by being high-contrast, structured, and typographically dense rather than quiet.

### 04 — Loud Signal
**Bold, maximalist, neo-brutalist.** A saturated blue hero band, hand-drawn-feeling yellow highlighter marks behind the wordmark, hot-pink and mint accents, thick black borders, and hard offset drop-shadows on every card and chip. Format badges sit as slightly rotated stickers. Personality: loud, playful, internet-native confidence — the biggest tonal swing in the set, proving trust can also be signaled through joy and craft rather than restraint or technical austerity.

### 05 — Quiet Paper
**Deliberately quieter than every other concept, including both round-1 dark themes.** Warm cream background, an upright (non-italic) editorial serif wordmark, hairline rules instead of boxes, no dashed borders, no colored chips — format names are just tracked-out text separated by middle dots. The single accent (a muted clay/terracotta) appears only as a dot, a hairline, and the word "Verified." The idle/result toggle is styled as plain underlined text links rather than a boxed tab switch. Personality: premium through negative space and typographic restraint — closest to a high-end hardware brand's product page.

### 06 — Bento Deck
**Structurally the odd one out**: a CSS-grid bento/dashboard layout instead of a single centered column. The hero splits into a headline tile and a stat tile; the tool panel is a full-width tile with a pill-shaped segmented toggle; a lower row of tiles carries the format badges plus two trust-signal tiles ("142 camera models supported," "No ads"). Palette is a neutral cream base with an indigo primary accent and soft pastel tile fills (peach, mint, sky). Personality: modern SaaS/product-dashboard energy — signals engineering rigor through information architecture rather than through console/mono styling.

## Range covered in round 2

Light editorial (03) · loud maximalist color (04) · quiet minimal premium (05) · structurally unusual bento grid (06) — deliberately spanning palette, density, and layout so the four don't converge on any single mood, and none of them re-tread round 1's dark-console territory.
